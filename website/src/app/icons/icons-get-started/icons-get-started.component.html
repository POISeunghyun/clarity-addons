Getting started with Clarity Addons Icons

To include clarity icons follow the instructions from the Clarity Icons - Get Started Guide.

Before you can use an icon, you must import if in your current component.ts file

You can also import all available icons at once

Usage of clarity addons icons is as easy as use the cds-icons tag with the desired shape

Information about support of icon badges

Support for the badge property in clarity addons icons is currently only available to:

  • customer-vip
  • repeat-repair
  • customer-waiting
  • replacement-vehicle

and is used as follows: The badge property is from the Clarity Design documentation's Icons-API. For Further reading, visit the official documentation.

To add badge support for other icons, the icons in question need to be extended from IconSvgString into IconShapeCollection in the shapes.ts file of this library.
For more details on how to implement it, either compare the currently supported icons with the non-supported icons or look into the official Clarity Design source code.
